## Problem

Government platforms (IRIS, Provincial Revenue Authority, Digital Invoicing) lacked centralized observability across distributed microservices, making debugging and analytics difficult at scale.

## Solution

Designed and deployed a centralized ELK (Elasticsearch, Logstash, Kibana) stack that:

- Processes **1TB+ logs** across multiple platforms
- Integrates with IRIS platform, Provincial Revenue Authority, and Digital Invoicing System
- Provides real-time log analytics and search capabilities
- Enables proactive monitoring and incident response

## Outcomes

Significantly improved observability and log analytics across distributed services, enabling faster debugging and better system health monitoring.

## Tech Stack

ELK Stack, Java 21, Spring Boot, Microservices, Kafka, Redis
