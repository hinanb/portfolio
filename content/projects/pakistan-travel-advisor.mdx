## Problem

Pakistan lacked a unified travel search platform aggregating packages, hotels, and itinerary planning across multiple providers.

## Solution

Built a full-stack travel platform with three core components:

- **Travel packages aggregator** — Showcased offerings from top agencies with company profiles
- **Hotel aggregator** — Listed top-rated accommodations with rental options from multiple booking platforms
- **ML itinerary planner** — Optimized hotel and flight selections based on trip duration and budget

Data was crawled from the web every 12 hours using BeautifulSoup cron jobs.

## Outcomes

- 10,000+ website visits
- 7,000+ active community members
- 50+ trips booked, 20 hotel bookings
- **Acquired in early 2022** by a Swedish entrepreneur

## Tech Stack

Python, BeautifulSoup, Machine Learning, Web Scraping, Cron Jobs
