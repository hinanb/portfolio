## Problem

During the COVID-19 pandemic, Pakistan needed data-driven forecasting to understand the trajectory of the virus spread.

## Solution

Deployed an ML-based web application that predicted Pakistan's COVID-19 trajectory 30 days ahead using historical data and machine learning models.

## Outcomes

Successfully forecast the curve flattening significantly ahead of official projections, demonstrating the value of predictive analytics in public health.

## Tech Stack

Python, Machine Learning, Web Application, Data Visualization
