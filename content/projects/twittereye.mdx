## Problem

Researchers and developers needed a simple tool to analyze and visualize public sentiment on Twitter topics at the keyword level.

## Solution

Published **twitterEye**, an open-source PyPI package that enables:

- Keyword-level reaction tracking
- Trend visualization
- Sentiment analysis on Twitter topics

## Impact

Available on PyPI for researchers and developers to integrate sentiment analysis into their workflows.

## Tech Stack

Python, NLP, PyPI, Data Visualization
